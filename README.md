# 🎮 Skilldex — Gamified Learning Platform

🚀 Skilldex is an interactive learning platform that enables users to **learn new skills through games**, making education engaging and fun while providing **performance tracking and progress insights**.

---

## 📌 Features

* 🎮 **Gamified Learning Experience**
  Learn concepts through interactive games instead of traditional methods.

* 📚 **Skill-Based Modules**
  Structured learning paths designed to build specific skills step-by-step.

* 📊 **Performance Tracking**
  Monitor progress and performance across different activities.

* 🏆 **Progress & Achievement System**
  Track milestones and improve consistency through measurable goals.

* 🧠 **Engaging User Experience**
  Combines learning with gameplay to increase retention and motivation.

## ⚡ How It Works

1. User selects a skill or learning module
2. Engages with game-based challenges
3. System evaluates performance in real-time
4. Progress and scores are stored and tracked
5. User can view improvement over time
